# 📖 EPUB Editor

Локальный веб-редактор метаданных EPUB-файлов. Работает в браузере, не требует интернета.

## Возможности

### Редактор
- Изменение названия и автора книги
- Установка обложки (JPG, JPEG, PNG, WEBP и другие форматы)
- Обрезка обложки по заданному соотношению сторон (2:3, 1:1, 16:9 и др.)
- Добавление текста на обложку с выбором шрифта, размера, цвета и позиции
- Поддержка кириллических шрифтов (TTF/OTF)

### Объединение
- Загрузка нескольких EPUB одновременно
- Изменение порядка перетаскиванием
- Объединение содержимого, оглавления (NCX + NAV), ресурсов и метаданных
- Результат — полностью рабочий EPUB-файл

## Установка

### Требования
- Python 3.9+
- pip

### Шаги

```bash
# 1. Клонировать репозиторий
git clone https://github.com/ваш-логин/epub-editor.git
cd epub-editor

# 2. Создать виртуальное окружение (рекомендуется)
python -m venv venv

# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Запустить
python app.py
```

### Открыть в браузере
```
http://localhost:5000
```

## Шрифты с кириллицей

Приложение поддерживает загрузку TTF/OTF-шрифтов через интерфейс. Если шрифт не загружен, используется встроенный DejaVu Sans.

Рекомендуемые бесплатные шрифты с кириллицей:
- **PT Serif / PT Sans** — https://fonts.google.com/specimen/PT+Serif
- **Raleway / Roboto** — https://fonts.google.com
- **Paratype** — https://www.paratype.ru

Скачайте TTF-файл и загрузите его через кнопку «Загрузить» в разделе шрифтов.

## API

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/upload` | Загрузить EPUB, получить метаданные |
| GET | `/api/cover/<file_id>` | Получить обложку |
| POST | `/api/process-cover` | Обработать изображение (обрезка + текст) |
| POST | `/api/save` | Сохранить изменённый EPUB |
| GET | `/api/fonts` | Список доступных шрифтов |
| POST | `/api/fonts/upload` | Загрузить шрифт |
| POST | `/api/merge/upload` | Загрузить файлы для объединения |
| POST | `/api/merge/execute` | Выполнить объединение |
| POST | `/api/cleanup` | Удалить временные файлы |
