# 📖 EPUB Editor

Браузерный редактор метаданных EPUB-файлов. Работает прямо в браузере — никакой установки.

🔗 **[Открыть редактор](https://твойлогин.github.io/epub-editor)**

---

## Возможности

### Редактор
- Изменение названия и автора книги
- Установка обложки (JPG, PNG, WEBP и другие форматы)
- Обрезка обложки по соотношению сторон с визуальным предпросмотром
- Добавление текста на обложку с выбором цвета, размера и позиции
- Загрузка TTF/OTF шрифтов с поддержкой кириллицы

### Объединение
- Загрузка нескольких EPUB одновременно
- Изменение порядка книг перетаскиванием
- Объединение содержимого, оглавления (NCX + NAV) и ресурсов
- Результат — полностью рабочий EPUB-файл

---

## Приватность

Все файлы обрабатываются **локально в браузере**. Ничего не отправляется на серверы.

---

## Технологии

- [JSZip](https://stuk.github.io/jszip/) — чтение и запись EPUB
- [Cropper.js](https://fengyuanchen.github.io/cropperjs/) — обрезка изображений
- [opentype.js](https://opentype.js.org/) — рендеринг TTF/OTF шрифтов на Canvas
- Canvas API — нанесение текста на обложку
